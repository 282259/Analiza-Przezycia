# Analiza-Przezycia
Sprawozdania koncentrują się na statystycznej analizie przeżycia, wykorzystującej dane kliniczne dotyczące czasu do wystąpienia remisji choroby. W ramach prac zaimplementowano szereg metod analizy danych cenzurowanych typu I oraz II, obejmujących zarówno podejście nieparametryczne, jak i modelowanie regresyjne w środowisku R. Kluczowym elementem analizy jest estymacja funkcji przeżycia za pomocą estymatora Kaplana-Meiera oraz porównywanie grup pacjentów leczonych różnymi preparatami przy użyciu testów log-rank, Gehana-Breslowa, Peto-Peto oraz Tarone-Ware. Projekt zawiera również dopasowanie rozkładów teoretycznych (wykładniczego, Weibulla oraz Rayleigha) metodą największej wiarygodności (MLE) oraz zaawansowaną analizę czynników ryzyka z wykorzystaniem modelu proporcjonalnych hazardów Coxa, w tym weryfikację istotności zmiennych takich jak wiek, płeć czy stan sprawności pacjenta.
